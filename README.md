# Devtools

A bunch of usefull tools. Designed by human. Made by codex. 

## CLI index

- `tokemon` (`bin/tokemon`): Report token usage from local Codex and Claude session logs; detailed usage in [`docs/usage/tokemon.md`](docs/usage/tokemon.md).
- `json_lint.py` (`bin/json_lint.py`): Validate JSON from a file path or stdin; detailed usage in [`docs/usage/json_lint.md`](docs/usage/json_lint.md).

## Docs layout

- `docs/usage/`: CLI usage docs (`{tool-name}.md`)
- `docs/specs/`: design and architecture specs
