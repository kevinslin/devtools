# Tools

Small local CLIs for day-to-day developer workflows.

## CLI index

- `tokemon` (`bin/tokemon`): Report token usage from local Codex and Claude session logs; detailed usage in [`docs/specs/tokemon/README.md`](docs/specs/tokemon/README.md).
- `json_lint.py` (`bin/json_lint.py`): Validate JSON from a file path or stdin.
